\LARGE

\noindent Draw axes with \(-2 \le x \le 6\) and \(-1 \le y \le 11\).
On these axes draw the following curves.

\begin{enumerate}
\item \(y = x^2 + 2 x\)
\item \(y = 2.25 + x - 0.25 x^2\)
\item \(y = x^2 + 2 x + 2.25\)
\item \(y = x^2 - 10 x + 24\)
\item \(y = x^2 - 10 x + 26.25\)
\item \(y = -0.2 + x - 0.25 x^2\)
\end{enumerate}
